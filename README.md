# TugasKu

TugasKu adalah aplikasi PWA manajemen tugas mahasiswa dengan fokus pada:

- pencatatan tugas kuliah,
- deadline-centered workflow,
- reminder bertingkat,
- pengelompokan berdasarkan mata kuliah,
- progres tugas,
- statistik produktivitas sederhana.

## Fitur utama

- Tambah tugas dengan mata kuliah, deadline, dosen, bobot nilai, estimasi durasi, tipe pengerjaan, dan kanal pengumpulan.
- Smart Priority berdasarkan deadline, bobot, estimasi durasi, status, dan prioritas manual.
- Filter tugas berdasarkan status, mata kuliah, pencarian, dan prioritas.
- Kalender deadline 7 hari.
- Ringkasan mata kuliah aktif.
- Arsip tugas selesai.
- Dukungan PWA: installable, cache offline, dan browser notification.

## Catatan reminder

Reminder lokal pada versi ini berjalan saat aplikasi dibuka, aktif, atau kembali ke foreground. Untuk push notification penuh di background dibutuhkan arsitektur tambahan seperti backend dan web push.
